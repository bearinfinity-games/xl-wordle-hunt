# xl-wordle-hunt
I'm an x-letter word, Guess Me!

A React + TypeScript x-letter Word Guessing Game.

## Features

- Exciting word guessing game with difficulty levels.
- On-screen QWERTY keyboard
- Letter color feedback
  - Orange -> valid letter in wrong position
  - Green -> valid letter in correct position
  - Wrong keys are progressively highlighted on the keyboard
- Word validation
